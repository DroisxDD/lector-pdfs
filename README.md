# Lector PDFs Mobile

Aplicación móvil desarrollada con Expo + React Native para lectura optimizada de PDFs con soporte de zoom y modo lectura accesible.

## Tecnologías

- Expo
- React Native
- TypeScript

## Objetivo V1

- Visualización de PDFs
- Soporte de zoom
- Experiencia de lectura clara

## Instalación

```bash
npm install
npx expo start
```
